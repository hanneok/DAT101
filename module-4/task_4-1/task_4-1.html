    2. Object-oriented development allows for better organization and management of code by putting them into different objects.

  2. 2. In Task 1, we create a static object called "AccountType." Explain what this object is used for and how this type of object can help us in software development.

    3. The static object ""AccountType" is used to define 4 different types of bank accounts in one place. This makes it easier to manage and maintain the code.

  3. 3. Discuss why we use encapsulation in classes. Why do we want to "hide" code within our class? Explain how we can create "private" and "public" members in a constructor function.

    4. Encapsulation is used to hide the internal details of a class and only expose what is necessary. This helps in maintaining the integrity of the data and prevents unauthorized access. you use a "#" in front of a property or method name to make it private.

  4. 4. Explain what happens when we use the "new" command in JavaScript.

    5. The "new" command is used on a function/class to create a new empty object and sets the constructor's 'this' to point to that object.
