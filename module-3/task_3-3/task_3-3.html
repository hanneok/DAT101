1. 1. What is the difference between invoking and creating a function?

    2. Invoking a function means to execute it, while creating a function just means to define it.

    3. Semicolons are used at the end of statements to indicate the end of it. Code blocks are used to group statements together, and are used in functions, loops, and conditionals.

    4. If a function doesn't return a value, it will result in "undefined" if you try to store its value.

  7. 7. Explain what happens when we use recursive functions.
